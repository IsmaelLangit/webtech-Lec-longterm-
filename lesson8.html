CHAPTER 8: Node JS Server Side Scripting

Node JS

NodeJS

Node.js – Has the same modules with JavaScript libraries. Use to create desktop applications. Node.js has a set of built-in modules which you can use without any further installation.

Node.js Built-in Modules version 6.10.0

  • assert – provides a set of assertion tests

  • buffer – used to handle binary data

  • child process – to run a child process

  • cluster – to split a single node process into multiple processes

  • crypto – use to handle OpenSSL cryptographic functions

  • dgram – use to provide implementation of UDP datagram sockets

  • dns – used to do DNS lookups and name resolution functions

  • domain – use to handle unhandled errors

  • events - use to handle events

  • fs – use to handle file system

  • http - use to make node.js act as an HTTP server

  • https – use to make node.js act as an HTTPS server

  • net – use to create server and clients

  • os – provides information about the operation system

  • path – use to handle file paths

  • punycode – use as a character encoding scheme

  • querystring- use to handle URL query strings

  • readline- use to handle readable streams one line at a time

  • stream – use to handle streaming data

  • string_decoder – use to decode buffer objects into strings

  • tls – use to implement TLS and SSL protocols

  • tty – use to provide classes used by a text terminal

  • url – use to parse URL strings

  • util – use to access utility functions

  • v8

  • vm – to compile JavaScript code in virtual machine

  • zlib – use to compress or decompress files
