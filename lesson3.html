CHAPTER 3: HTML (HyperText Markup Language)

HTML

HTML (Hypertext Markup Language) is used to create web pages. A web page is an electronic document. According to MDN or Mozilla Developer network, HTML is the most basic building block of the Web.

HTML vs. CSS vs. JavaScript

The purpose of HTML is for the content and structure of the web page. To bring the content to life, CSS may be used. CSS is where a web page can be made to be aesthetic and presentable. To interact with the web page, JavaScript may be used. JavaScript is used for the behavioral aspects of the web page.
