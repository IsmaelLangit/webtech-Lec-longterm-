CHAPTER 5: Javascript

JAVASCRIPT

EXTERNALLY LINKED SCRIPT
  • Gets part of an array to another location on the array. And returns the array (with the same length)

  • Syntax:
    • <script src="myscripts.js"></script>
EMBEDDED SCRIPT

  • Between a pair of and </script> tags

  • From an external file specified by the src attribute of a <script> tag

  • Syntax: 
    • <script>
      document.getElementById("demo").innerHTML = "My First JavaScript";
      </script>
INLINE SCRIPT
  • Normally the HTML event attributes are used to call functions specified elsewhere in script tags.
  • Syntax: 
    • <p><a href="#" onClick="alert('Hello');">Click Me</a></p>
TOP LEVEL / GLOBAL
  • _dump

    Prints a string representation of an object to the log file.

  • eval

    Evaluates an expression or statements.

  • isNaN

    Determines whether the result of a string parsed by parseInt or parseFloat is a valid number.

  • parseFloat

    Parses a string and returns a floating point number.

  • parseInt

    Parses a string and returns an integer of the specified radix or base.

  • print

    Prints a string to the log file.

FUNCTION
  • A JavaScript function is defined with the function keyword, followed by a name, followed by parentheses ().
  • Syntax:
    • function name(parameter1, parameter2, parameter3) {
          code to be executed}

Mutator Methods

Modifies the target array

Array.prototype.copyWithin()

Gets part of an array to another location on the array. And returns the array (with the same length).

Syntax:

arr.copyWithin(target)

arr.copyWithin(target, start)

arr.copyWithin(target, start, end)

Example:

var sample = ['ichi' 'san', 'yon', 'go', 'ni'].copyWithin(2, 0);

console.log(sample);

// Result : ["ichi", "san", "ichi", "san", "yon"]

Gets part of an array to another location on the array. And returns the array (with the same length).

Syntax:

Array.prototype.fill()

Fills an array with elements from the start to the end index with a static value.

Syntax:

arr.fill(value)

arr.fill(value, start)

arr.fill(value, start, end)

Example:

var sample = [8, 9, 3]

sample.fill(10);

//Result: [10, 10, 10]

Array.prototype.pop()

Removes the last element of an array.

Syntax:

array.pop()

Example:

var sample = [‘cat’, ‘dog’, ‘fish’];

sample.pop();

// Results: [‘cat’, ‘dog’]

Array.prototype.push()

Adds one or more elements to end of an array.

Syntax:

array.push([element1[, ...[, elementN]]])

Example:

var sample = [‘a’, ‘b’, ‘c’];

sample.push(‘d’, ‘e’);

//Results: [‘a’, ‘b’, ‘c’, ‘d’, ‘e’)

Array.prototype.splice()

And and/or removes elements from an array

Syntax:

array.splice(start)

array.splice(start, deleteCount)

Accessor Methods

Joining an array to another array or a value.

Syntax:

var new_array = old_array.concat(value1[, value2[, ...[, valueN]]])

Example:

Iterator Methods

Returns a new array iterator object that contains the key/value pairs for each index of the array.

Syntax:

array.entries()

Example:

var sample = [‘dog’, ‘cat’, ‘duck’];

var iterator = sample.entries();

for (let x of iterator) {

console.log(x);

}

// Results :[0, 'dog']

// Results :[1, 'cat']

// Results :[2, 'dog']

Array.prototype.every()

Validates all elements inside an array if it satisfies the function, true if valid, and false otherwise.

Syntax:

array.every(callback[, thisArg])

Example:

function isEven (element, index, array) {

return element%2 = 0;

}

[1, 2, 3, 4, 5].every(isEven ); // Result : false

[12, 22, 32, 42, 52].every(isEven ); // Result : true

Array.prototype.filter()

Syntax:

Example:

function isOdd(element, index, array) {

return element%2 != 0;

}

[10, 11, 12, 13, 14].filter(isOdd );

// Result : [11,13]

OBJECT

Use the prototype object oriented

Creating an Object

  • The same as Javascript variables, it is also case sensitive. On defining a property of an object, just assign it with a value.
  • Example:

var flower= new Object();

flower.name = ‘rose’;

flower.color = ‘red’

flower.description = ‘beautiful’;

Deleting properties from an Object

  • By using the delete operator, we can remove a property from an Object.

var flower= new Object();

flower.name = ‘rose’

flower.color = ‘red’;

flower.description = ‘beautiful’;

delete flower.color; // returns “true”

console.log (flower);

//Result : Object {name: "rose", description: "beautiful"}

Using JavaScript Object Notation (JSON)

  • Used for storing data.
  • An object can composed of an array, set of objects, an array inside an array or an object, or an object inside an array or another object. Data storing could be complex.
  • Example of JSON file:

flower= [

{

"name" : "Rose",

"color" :"red",

"description" : “beautiful”

},

{

"name" : "Rose",

"color" :"red",

"description" : “beautiful”

}

]

Storing JSON on a local storage

  • By using localStorage property, we can store and retrieve a local storage object

    • var sampleJSON = JSON.stringify(flower);

    localStorage.setItem(“data”, sampleJSON);

  • Example of retrieving data:

    • var retrievedJSON = localStorage.getItem("data");

    var flower = JSON.parse(retrievedJSON);
