CHAPTER 5: Javascript

JAVASCRIPT

EXTERNALLY LINKED SCRIPT
  • Gets part of an array to another location on the array. And returns the array (with the same length)

  • Syntax:
    • <script src="myscripts.js"></script>
EMBEDDED SCRIPT

  • Between a pair of and </script> tags

  • From an external file specified by the src attribute of a <script> tag

  • Syntax: 
    • <script>
      document.getElementById("demo").innerHTML = "My First JavaScript";
      </script>
INLINE SCRIPT
  • Normally the HTML event attributes are used to call functions specified elsewhere in script tags.
  • Syntax: 
    • <p><a href="#" onClick="alert('Hello');">Click Me</a></p>
TOP LEVEL / GLOBAL
  • _dump

    Prints a string representation of an object to the log file.

  • eval

    Evaluates an expression or statements.

  • isNaN

    Determines whether the result of a string parsed by parseInt or parseFloat is a valid number.

  • parseFloat

    Parses a string and returns a floating point number.

  • parseInt

    Parses a string and returns an integer of the specified radix or base.

  • print

    Prints a string to the log file.

FUNCTION
  • A JavaScript function is defined with the function keyword, followed by a name, followed by parentheses ().
  • Syntax:
    • function name(parameter1, parameter2, parameter3) {
          code to be executed}
ARRAY
Some Characterictics
  • Allow mixed elements.

Ex. array1 = [10, true, 'hello'];

  • Allow to change the size of the array

Ex. var array1 = new Array(5);

array1.length = 4;

  • Array indices can be contiguous

Ex. var array1 = [1, 2, 3, 4, 5];

array1[8] = 8;

console.log(array1);

//Result : [1, 2, 3, 4, 5, undefined, undefined, 8]

Mutator Methods

Modifies the target array

Array.prototype.copyWithin()

Gets part of an array to another location on the array. And returns the array (with the same length).

Syntax:

arr.copyWithin(target)

arr.copyWithin(target, start)

arr.copyWithin(target, start, end)

Example:

var sample = ['ichi' 'san', 'yon', 'go', 'ni'].copyWithin(2, 0);

console.log(sample);

// Result : ["ichi", "san", "ichi", "san", "yon"]

Array.prototype.fill()

Fills an array with elements from the start to the end index with a static value.

Syntax:

arr.fill(value)

arr.fill(value, start)

arr.fill(value, start, end)

Example:

var sample = [8, 9, 3]

sample.fill(10);

//Result: [10, 10, 10]

Array.prototype.pop()

Removes the last element of an array.

Syntax:

array.pop()

Example:

var sample = [‘cat’, ‘dog’, ‘fish’];

sample.pop();

// Results: [‘cat’, ‘dog’]
